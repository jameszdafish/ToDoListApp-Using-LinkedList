# To-Do List App Using Linked List

A desktop to-do list application built with Python and Tkinter, 
using a custom-built linked list as the underlying data structure 
for task management.

## Technologies Used

- Python
- Tkinter
- Custom Linked List implementation

## Features

- Create up to 10 tasks with a character limit per entry
- Each task includes a due date and time
- Checkbuttons to mark tasks as complete
- Add, edit, and delete tasks

## Setup & Installation

1. Clone the repository
2. Run the app:
   python main.py
